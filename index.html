This is the accompanying website of the book "Displaying time series, spatial and space-time data with R" to be published with Chapman&Hall/CRC.

This book will provide methods to display time series, spatial and space-time data using R. The code of each method will be detailed and commented through practical examples with real data. This website publishes the code and main graphics of each chapter.
